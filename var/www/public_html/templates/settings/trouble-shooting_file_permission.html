Datei- und Ordnerberechtigungen {function="showGoBackIcon('?s=settings&do=trouble-shooting')"}
Möchtest du wirklich fortfahren? Folgende Befehle werden ausgeführt:
- sudo chown -R {$file_permission_whoaim}:{$file_permission_whoaim} {$file_permission_absolute_path}/
- sudo find {$file_permission_absolute_path}/ -type d -exec chmod 755 {} +
- sudo find {$file_permission_absolute_path}/ -type f -exec chmod 644 {} +
